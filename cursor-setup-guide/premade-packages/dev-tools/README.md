# Development Tools

Development tools and utilities for managing projects and workflows.

## Available Tools

1. **[CLI Menu](./cli-menu/)**
   - Interactive command-line menu system
   - Arrow key navigation
   - Easy integration with your tools
   - Perfect for creating launcher scripts

2. **[Config Manager](./config-manager/)**
   - Handle YAML/JSON configuration files
   - Safe config merging
   - Version management
   - Default value handling

## Use Cases

- Create launcher menus for your automation tools
- Manage project configurations
- Build interactive CLI applications
- Organize and access multiple tools from one place
