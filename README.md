# Development Environment Setups

Complete setup guides for various development environments and tools.

## Available Setups

- **[Cursor IDE Setup Guide](./cursor-setup-guide/)** - Complete setup guide for Cursor IDE with dependencies, extensions, and premade packages

## Adding New Setups

To add a new setup guide:
1. Create a new folder in `setups/` (e.g., `vscode-setup-guide/`)
2. Follow the structure of `cursor-setup-guide/` as a template
3. Update this README with a link to your new setup

## Structure

Each setup guide should include:
- Main README with installation instructions
- Platform-specific instructions (Windows, macOS, Linux)
- Required extensions/tools
- Premade packages or code examples
- Quick reference guide
- Installation scripts
